# 第二章简介

　　本章将详细介绍简体GB2312手写字体的制作过程，从模板纸的设计、书写笔的选取、字稿的扫描方法等入手，直到结合各种软件切图、重命名、导入、生成等，过程相当复杂，我介绍的这一套流程是我经过反反复复调试的结果，在字体创作的过程中我走了很多弯路。有一丁点错误就会导致整个制作过程中断，并且很难从网上搜索到处理问题的方法。希望大家认真阅读并一定按照流程按部就班地进行创作。
